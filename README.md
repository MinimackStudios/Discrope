# DiskChat

[![License: GPL v3](https://img.shields.io/badge/License-GPLv3-blue.svg)](LICENSE)

A free chat platform built as a no-nonsense alternative to Discord: no invasive data collection, no paywalls, no Nitro.

## Why DiskChat?

Discord has become the default community platform, but it comes with real tradeoffs: invasive data collection, aggressive monetization, and an ever-growing list of features locked behind a subscription. DiskChat is built on the belief that real-time communication should just be free, with every feature available to every user from day one.

## Features

DiskChat covers all of Discord's essential day-to-day features: servers, channels, direct messages, file attachments, reactions, friend requests, and more. It also features a few Nitro-exclusive features for free. More features may be added over time.

> [!WARNING]
DiskChat is actively being developed and may not be as stable or polished as established chat platforms. Bugs, rough edges, and missing features should be expected. Use at your own discretion.
